◆ colorNormalize()

RgbColor8_t colorNormalize ( RgbColor16_t  sourceColor)

Returns normalized 8 bit RGB color based on provided 16bit RAW RGB value and defined black and white reference colors.

Following preprocessor values are used for calculations
#define COLOR_NORMAL_RESULT_BLACK_LEVEL as normalization result for black referent level
#define COLOR_NORMAL_RESULT_WHITE_LEVEL as normalization result for white referent level

Parameters
sourceColorSource 16bit integer RGB color from the sensor that needs to be normalized
Returns
normalized Normalized color returned as 8bit integer RGB

◆ colorSetModels()

void colorSetModels ( const RgbColor8_t *  colorModels,
uint8_t  colorModelsAvailable 
)

Defines array of color models to be used in colorFindNearest function.

Function does not check the integrity or array size so the array needs to have at least #colorModelsAvailable number of elements.
NOTE!!! Only the reference of the array is stored in the internal structures for saving memory, so these values should not be altered after setting it here.

Parameters
colorModelsArray of RGB color models with number of elements at least equal colorModelsAvailable.
colorModelsAvailableNumber of colors in the colorModels array.
