void settingsInit ( void  )

Initializes routines and variable responsible for button operations.

Following definitions to be set in config.h file
#define KMSG_MAGIC string that defines module and version allowing to preserve and update settings between versions (e.g. "KMSG100")
#define KMSG_MAGIC_LENGTH number of bytes in magic string defined in KMSG_MAGIC (e.g. 8 in above example last byte is '\0' so it's 8 bytes)
NOTE: To preserve EEPROM settings make sure EESAVE fuse bit is correctly defined (EESAVE = 0)
In this version once EEPROM is erased or has different magic value than defined in KMSG_MAGIC then values are overwritten with default presets 1 to 4
