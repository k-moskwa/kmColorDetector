◆ timer1RegisterCallback()

void timer1RegisterCallback ( Timer1Callback *  callback,
void *  userData 
)

Register callback for the Timer1.

In order to make it working explicit call to timer1Start and timer1EnableInterrupt need to be done from program routines. NOTE: The callback is issued within interrupt routine and is not buffered, so it should be as short as possible.

Parameters
callbackPointer to the callback function to be issued on the interrupt.
userDataUser data void pointer to structure to be delivered to callback "as-is".
